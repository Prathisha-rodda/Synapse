# Synapse Backend

The backend will be built with Java and Spring Boot.

## Planned Features

- User registration
- User login
- JWT authentication
- Career path APIs
- Project APIs
- Roadmap APIs
- Task progress APIs
- Dashboard summary API

## Planned Package Structure

```text
com.synapse.platform
  auth
  user
  careerpath
  project
  roadmap
  progress
  dashboard
  common
  config
```

## First Build Tasks

1. Create Spring Boot project.
2. Add Spring Web, Spring Security, Spring Data JPA, MySQL Driver, and Validation.
3. Configure MySQL connection.
4. Create user entity and repository.
5. Build signup and login APIs.
6. Add JWT authentication.
