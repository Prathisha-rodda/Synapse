# Project Overview

## Name

Synapse

## One-Line Description

Synapse is a project-based developer learning platform that turns career goals into structured, real-world build plans.

## Vision

The long-term vision is to help developers become interview-ready and job-ready by learning through real projects, guided modules, progress tracking, and eventually AI-powered mentoring.

## Target Users

- Working developers who want to upskill consistently
- Students who want practical project experience
- Developers preparing for full-stack interviews
- Self-learners who need structure and accountability

## V1 User Journey

1. User creates an account.
2. User selects the Java Full Stack Developer path.
3. User chooses the Expense Tracker project.
4. User enters experience level, target duration, and weekly study hours.
5. Synapse shows a module-based learning roadmap.
6. User opens a module and reviews the concepts required.
7. User completes implementation tasks.
8. Synapse tracks progress and shows dashboard metrics.

## V1 Success Criteria

V1 is successful when a user can:

- Sign up and log in
- View one career path
- Select one recommended project
- See project modules and tasks
- Mark tasks as completed
- See progress on a dashboard

## Out of Scope for V1

- AI mentor
- AI roadmap generation
- AI mock interviews
- Resume review
- GitHub activity tracking
- Multiple career paths
- Payment system
- Admin panel
