# Learning Roadmap

This roadmap is for building Synapse while improving Java full-stack skills.

## Phase 1: Project Foundation

Goal: Create a clean GitHub project and understand the MVP.

Tasks:

- Create repository
- Add README
- Add project docs
- Decide V1 scope
- Define database model
- Define APIs

Skills learned:

- Product planning
- Technical documentation
- GitHub project presentation

## Phase 2: Backend Foundation

Goal: Create the Spring Boot backend.

Tasks:

- Create Spring Boot project
- Add dependencies
- Connect MySQL
- Create user entity
- Create auth APIs
- Add JWT authentication

Skills learned:

- Spring Boot
- REST APIs
- Spring Security
- JWT
- MySQL
- JPA

## Phase 3: Frontend Foundation

Goal: Create the React frontend.

Tasks:

- Create React app with Vite
- Add routing
- Create login page
- Create signup page
- Connect auth APIs
- Protect private routes

Skills learned:

- React
- TypeScript
- Forms
- Routing
- API integration

## Phase 4: Roadmap Experience

Goal: Build the core learning experience.

Tasks:

- Seed Java Full Stack path
- Seed Expense Tracker project
- Build roadmap API
- Build roadmap page
- Build module details page
- Show concepts and tasks

Skills learned:

- Database relationships
- API response modeling
- UI composition
- Frontend data fetching

## Phase 5: Progress Tracking

Goal: Let users track completed tasks.

Tasks:

- Add user task progress model
- Build complete task API
- Build reopen task API
- Show progress percentage
- Build dashboard summary API
- Build dashboard UI

Skills learned:

- User-specific state
- Backend calculations
- Dashboard design
- Full-stack workflows

## Phase 6: Deployment

Goal: Make Synapse available online.

Tasks:

- Prepare production build
- Deploy frontend
- Deploy backend
- Configure production database
- Add environment variables
- Update README with live links

Skills learned:

- Deployment
- Environment configuration
- Production readiness
