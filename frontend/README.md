# Synapse Frontend

The frontend will be built with React, TypeScript, and Vite.

## Planned Pages

- Login
- Signup
- Dashboard
- Career Path Selection
- Project Selection
- Roadmap
- Module Details

## Planned Structure

```text
frontend/src/
  api/          API client functions
  components/   Reusable UI components
  features/     Feature-specific UI and logic
  layouts/      Page layouts
  pages/        Route-level pages
  routes/       App routing
  types/        TypeScript types
```

## First Build Tasks

1. Create Vite React app.
2. Add React Router.
3. Create login and signup pages.
4. Add API client.
5. Build protected dashboard route.
